# 简易 H5 记事本

使用 Next.js 脚手架创建。

## 在线体验

已部署到 Github Pages，可在线访问。地址 => [https://maplor.github.io/m-note/](https://maplor.github.io/m-note/)

## 本地启动

`Node.js` 版本要求 `18.17.0+`，使用以下命令启动

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000) 看到本地启动效果。
